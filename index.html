AI Interview Prep Assistant

A full-stack Next.js app for practicing technical interviews. On the backend I set up a RAG pipeline that chunks and embeds a knowledge base into Pinecone, then pulls the closest matches into each Gemini 2.5 Flash prompt so the answers stay grounded. Responses stream in token by token so the chat feels live.

Multimodal AI Pantry & Recipe Generator

An inventory app built with Next.js and Firebase. Users can add pantry items by typing or by uploading a photo, and Gemini's vision model reads the image, identifies the food, and adds it to the list on its own. Items and their quantities live in Firestore, and when you ask for recipes the app sends your current ingredients to Gemini and renders whatever it comes back with.

  • Next.js

  • Gemini Vision
  • Firestore
  • GitHub

SparkCards: AI Flashcard SaaS

A flashcard SaaS built on Next.js that generates a 10-card study deck from whatever text you give it, using the Gemini API. The app prompts the model for clean JSON and parses it carefully, so a bad response won't crash the page. Clerk takes care of authentication, Firestore saves each user's decks in a single batched write, and Stripe Checkout handles the $5/month Pro plan. Saved cards show up in a flip-card study view.
